## 🔔 **프로젝트 소개**

파리에 관련된 여행 정보와 일정을 자세하게 공유하는 사이트입니다.
실제로 파리 여행을 준비했을 떄 느꼈던 정보 부족의 어려움을 토대로 제작하게 되었습니다.
일정표, 숙소, 교통, 관광지 등 계획과 후기를 통해 자세한 정보를 쉽게 얻을 수 있고 자신의 추억을 기록해 두는 효과도 기대 할 수 있는 사이트입니다.
